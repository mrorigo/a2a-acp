Quick Start

Follow the README quick start: install, point at your ACP agent, configure credentials, launch the gateway, and ship a JSON-RPC message.

git clone https://github.com/mrorigo/a2a-acp.git cd a2a-acp uv sync && uv pip install -e . export A2A_AGENT_COMMAND="/usr/local/bin/codex" export A2A_AGENT_API_KEY="${OPENAI_API_KEY}" export A2A_AUTH_TOKEN="your-secret-token" make run

Run the JSON-RPC demo and validate it with make test (and the governance-focused pytest slice) once you have tooling and policies wired up.

Instantly tap into the protocol

Run the server locally or in your cloud environment, then deliver messages through familiar JSON-RPC over HTTP.

curl -X POST http://localhost:8001/ \\ -H "Content-Type: application/json" \\ -H "Authorization: Bearer $A2A_AUTH_TOKEN" \\ -d '{ "jsonrpc": "2.0", "method": "message/send", "id": "msg_001", "params": { "message": { "role": "user", "parts": [{"kind": "text", "text": "Hello from the A2A mesh"}], "messageId": "msg_001" }, "metadata": {"agent_name": "codex-acp"} } }'
